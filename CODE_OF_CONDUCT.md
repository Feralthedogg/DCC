# Code of Conduct

## Our Standard

Contributors and maintainers must provide a respectful, harassment-free
environment. Be constructive, assume good faith, accept technical correction,
and focus criticism on code and decisions rather than people.

Unacceptable behavior includes harassment, threats, discrimination, sexualized
attention, deliberate intimidation, publishing another person's private
information, or sustained disruption of project work.

## Enforcement

Report conduct concerns privately to the repository maintainers. Maintainers
may edit or remove contributions, comments, and issues; issue warnings; impose a
temporary restriction; or permanently remove participation when needed. Reports
will be handled as confidentially as practical, with conflicts of interest
avoided where possible.

This policy applies in repository spaces and when someone is publicly
representing the project.
